## Getting Started

1. setup environment:
   1. corepack enable
      ```bash
      corepack enable
      ```
   2. pnpm package install
      ```bash
      pnpm install
      ```
2. run the development server:
    ```bash
    pnpm run dev
    ```
3. open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
